## Get started

1. Clone Repository
    ``` shell
    $ cd ~/catkin_ws/src
    $ git clone https://github.com/UTHAI-Humanoid/UTHAI-Tools.git
    ```

1. Edit .sk file
    ``` shell
    $ gedit uthai_kinematics.sk
    ```

1. Build sk to png
    ``` shell
    $ ./sk2png uthai_kinetics.sk
    ```
    จะได้ไฟล์รูปภาพ .png มา
